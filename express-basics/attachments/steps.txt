1. create a folder
2. npm init -> package.json
3. install express - npm i express@4 -> we can see express under dependencies section 
4. create a js file for maintaining express based works - app.js
5. create the server instance
    const express = require('express')
6. access express methods
    const app = express()
7. start the server by assigning the port
    app.listen( port , () => { } )
8. start creating different http methods 
    - app.get()
    - app.post()
9. test these methods using postman
